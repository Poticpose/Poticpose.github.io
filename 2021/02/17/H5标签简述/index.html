H5标签简述

H5标签简述

二月 17, 2021
  • 前端

P.S. NEW 代表H5的新添加

H5中不支持标签、主流浏览器不支持标签不再出现

标签

基础标签

  • <!DOCTYPE> ：定义文档类型

  • <html>：定义一个HTML文档

  • <title>：为文档定义一个标题

  • <body>：定义文档主体

  • <h1> to <h6>：定义文档内部内容标题

  • <p>：段落

  • <hr>：定义一个水平线

  • <!–…–>：声明一个注释

  • <br>：换行

HTML 元素以开始标签起始，
HTML 元素以结束标签终止

格式标签

  • <abbr title=”xxxxxxxxx”>：定义缩写

  • <address>：定义文档拥有者联系信息

  • <b>：定义粗体文字

  • <bdi> NEW ：允许您设置一段文本，使其脱离其父元素的文本方向设置。

  • <bdo dir=”rtl”>：定义文本方向（该dir属性表示文本从右向左）

  • <blockquote cite=””>：定义块引用，cite属性内为被引用信息来源

  • <cite>：定义引用

  • <code>：定义代码文本

  • <del>：定义被删除的文本

  • <dfn>：定义项目文本

  • <em>：定义强调文本

  • <i>：定义斜体文本

  • <ins>：定义被插入文本

  • <kbd>：定义键盘文本

  • <mark> NEW ：定义带有记号的文本

  • <meter value=”” min=”” max=””> NEW ：定义一个度量衡，，value属性当前值，max为最大值，min为最小值（仅限于已知最大、最小值的度量）

  • <pre>：定义预格式文本

  • <progress value=”” max=””>：定义一个进度条，value属性当前值，max为最大值

  • <q>：定义一个短引用

  • <ruby> NEW ：定义一个ruby注释，如中文注音或字符

  • <rt> NEW ：定义中文注音或字符的解释或发音

  • <rp> NEW ：定义不支持ruby标签的浏览器所显示的内容

  • <s>：定义加删除线的文本

  • <samp>：定义计算机代码样本

  • <small>：定义小号文本

  • <strong>：定义更加强烈的强调文本

  • <sub>：定义下标文本

  • <sup>：定义上标文本

  • <time datetime=””> NEW ：定义一个日期或时间，datetime属性内为time内容的时间信息

  • <u>：定义下划线文本

  • <var>：定义文本的变量部分

  • <wbr>：规定在文本中的何处适合添加换行符

表单标签

  • <form>：定义一个HTML表单用于用户输入

  • <input type=”” name=””>：定义一个输入控件，属性type是输入内容格式信息，name是输入内容所代名称

  • <textarea rows=”” cols=””>：定义多行的文本输入，属性rows是行数，cols是列数

  • <button type:”button”>：定义了一个按钮

  • <select>：定义了一个选择列表（下拉列表）

  • <optgroup>：定义选择列表中相关选项的组合

  • <option>：定义选择列表中的选项

  • <label>：定义input标签的标注

  • <fieldset>：定义一个围绕表单中元素的边框

  • <legend>：定义fieldset标签的标题

  • <datalist> NEW ：定义了input元素的可能的选项列表

  • <output name=”” for=””> NEW ：定义一个计算结果，属性name中的值代表该标签所获得的结果，for为参与结果运算的元素的id，可通过oninput属性在父级标签中定义运算

框架标签

  • <iframe src=””>：定义了一个内联框架，属性src指向框架内容的索引

图像标签

  • <img src=””>：定义一个图像，属性src指向图片地址

  • <map>：定义图像映射，指定可点击区域

  • <area>：定义图像内部的一个可点击区域

  • <canvas> NEW ：通过脚本（JavaScript）来绘制一个图像

  • <figure> NEW ：用于对元素进行组合

  • <figcaption> NEW ：定义一个figure标签内元素的标题

音频标签

  • <audio> NEW ：定义声音

  • <source> NEW ：定义media元素（<video>和<audio>）的媒体资源

  • <track default kind=”captions” srclang=”en” src=””/> NEW ：为媒体元素定义外部文本轨道，属性default提供默认形式，kind内为track提供的文本类型，srclang为文本语言，src为文本地址

  • <video> NEW ：定义一个音频或视频

链接标签

  • <a href=”xxxxx”>：定义一个页面超链接

  • <link rel=”stylesheet” type=”text/css” href=”XXXX.css”>：定义文档与外部资源关系，通常链接css文件

  • <main>：定义文档的主体内容

  • <nav>：定义导航链接

列表标签

  • <ul>：定义一个无序列表

  • <ol>：定义一个有序列表

  • <li>：定义一个列表项

  • <dl>：定义一个定义列表

  • <dt>：定义一个定义列表中的项目

  • <dd>：定义一个定义列表项目中的描述

  • <menu>：定义菜单列表

表格标签

  • <table>：定义一个表格

  • <caption>：定义表格标题

  • <th>：定义表格的表头单元格

  • <tr>：定义表格中的行

  • <td>：定义表格中的单元

  • <thead>：定义表格中的表头内容

  • <tbody>：定义表格中的主体内容

  • <tfoot>：定义表格中的脚注内容

  • <colgroup>：定义表格中供格式化的列组

  • <col span=”” style=””>：定义表格中一个列或多个列的属性值，属性span内为需定义的列数

样式/块标签

  • <style>：定义文档的样式信息
  • <div>：定义文档中的一个区块
  • <span>：对行内元素进行组合
  • <header> NEW ：定义文档的页眉部分（多为导航栏或内容介绍）
  • <footer> NEW ：定义文档的页脚部分
  • <section> NEW ：定义了文档的一个内容区域
  • <article> NEW ：定义文档中的独立内容
  • <aside> NEW ：定义article标签外的内容，需与周边内容相近
  • <details> NEW ：定义了用户可见的或者隐藏的需求的补充细节，通常为折叠状态
  • <dialog> NEW ：定义了一个对话框或者窗口
  • <summary> NEW ：定义了一个可见的标题，当用户点击时显示详细信息

元信息标签

  • <head>：定义文档所包含的信息，包容所有头部元素
  • <meta>：定义文档元信息，如关键词、作者、编辑时间等等
  • <base>：定义页面中所有链接的默认地址

程序标签

  • <script>：定义客户端脚本
  • <noscript>：定义针对不支持客户端脚本的用户的替代内容
  • <embed> NEW ：定义了一个容器，用来嵌入外部应用或者互动程序（插件）
  • <object>：定义了一个嵌入对象，以此向文档中添加多媒体内容，在无法解析状态下可现实中间代码
  • <param>：定义object的参数

全局属性

  • accesskey：设置访问元素的键盘快捷键。
  • class：规定元素的类名
  • contenteditable NEW ：规定是否可编辑元素的内容
  • data-* NEW ：用于存储页面的自定义数据
  • dir：设置元素中内容的文本方向。
  • draggable NEW ：指定某个元素是否可以拖动
  • hidden NEW ：hidden 属性规定对元素进行隐藏
  • id：规定元素的唯一id
  • lang：设置元素中内容的语言代码
  • spellcheck NEW ：检测元素是否拼写错误
  • style：规定元素的行内样式
  • tabindex：设置元素的 Tab 键控制次序
  • title：规定元素的额外信息
  • translate NEW ：指定是否一个元素的值在页面载入时是否需要翻译
